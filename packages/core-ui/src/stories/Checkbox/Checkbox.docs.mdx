# Checkbox

Use checkboxes for independent boolean choices. Add `label` and `description` for clarity.

## Usage
```tsx
import { Checkbox } from '@vsaithal/core-ui'

export function Example() {
  return <Checkbox label="Accept terms" />
}
```
