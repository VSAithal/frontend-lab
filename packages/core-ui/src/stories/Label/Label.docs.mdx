# Label

Labels provide accessible text for form controls. Pair with inputs using `htmlFor` and `id`.

## Usage
```tsx
import { Label, Input } from '@vsaithal/core-ui'

export function Example() {
  return (
    <div>
      <Label htmlFor="email">Email</Label>
      <Input id="email" placeholder="you@example.com" />
    </div>
  )
}
```
