# Button

Buttons are used for primary actions in a flow. Use the `variant`, `size`, and `rounded` props to control style and shape.

## Usage
```tsx
import { Button } from '@vsaithal/core-ui'

export function Example() {
  return (
    <Button variant="primary" size="base">
      Save changes
    </Button>
  )
}
```
